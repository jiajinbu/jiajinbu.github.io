Hexo github搭建博客

mac下利用hexo在Github上搭建博客

#注册Github账户
去github官网https://github.com注册即可。

#安装hexo
需要先安装npm。npm可从https://nodejs.org/en/直接下载OS X安装包。也可以安装ruby后安装brew，再用brew install node安装npm。
安装npm后，用
npm install -g hexo
cd /Users/bu/Documents/github_pages/website/hexo #写一个hexo文档所在的位置，网站存储在这里面
hexo init
npm install
安装hexo。
此时可以启动本地服务器
hexo server（或hexo s）
根据提示在浏览器打开http://0.0.0.0:4000即可。

#安装git
以前已经安装过git，因此这里没有安装。具体方法可搜索网上资料。

#配置SSH
需要利用ssh-keygene生成秘钥，如果以前已经存在连接到其他服务器的密钥，需要注意不要把以前的密钥覆盖，同时修改~/.ssh/下config文件，告诉每个服务器对应的密钥。

cd ~/.ssh
mkdir key_backup
cp id_rsa* key_backup #拷贝以防覆盖
ssh-keygene -t ras -C “xxxx@xxxx.com” #注册github的邮箱。该命令会提示私钥产生的位置，可以写/Users/bu/.ssh/github_rsa

修改~/.ssh/config文件（如果不存在则创建），按下面示例告诉每个服务器对应的密钥。第一个是保存以前的密钥和对应的服务器ip，第二个是连接github.com用的，只需要注意密钥文件地址IdentityFile设定的地址正确即可。

将SSH 公钥添加到Github
登录到GitHub页面，Account Settings->SSH Public Keys->Add another key将生成的key(id_rsa.pub文件）内容copy到输入框中，save。

测试连接
ssh git@github.com
输出如下结果即可：
Hi xxxxxxxxx! You’ve successfully authenticated, but GitHub does not provide shell access.

设置个人信息
git config –global user.name “xxxxxx”
git config –global user.email xxxxxx@xxx.com

#将hexo于github连接
修改hexo根目录（上文中的/Users/bu/Documents/github_pages/website/hexo）下_config.yml文件。（xxxx为你的github账户名称）

将博客上传到github
hexo deploy （或hexo d）
